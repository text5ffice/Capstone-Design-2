<h1 align="center">AI 기반 분전반·분전함 점검 및 매뉴얼 안내 시스템</h1>

<p align="center">
  분전반 및 분전함 내부를 촬영하면 AI가 부품과 이상 상태를 인식하고,<br>
  문제점, 주의사항, 매뉴얼 정보를 즉시 안내하는 현장 점검 보조 시스템
</p>

<hr>

<h2>📌 프로젝트 개요</h2>

<p>
분전반과 분전함은 외부에서 들어온 전력을 건물 내 여러 회로로 안전하게 분배하고,
과부하나 단락 발생 시 전원을 차단하여 설비를 보호하는 전기설비이다.
본 프로젝트는 작업자가 분전반 또는 분전함을 열고 내부 사진을 촬영하면,
AI가 구성 요소를 인식하고 이상 여부를 판별한 뒤 관련 매뉴얼과 점검 정보를 즉시 제공하는 시스템을 목표로 한다.
</p>

<h2>🎯 프로젝트 목표</h2>

<ul>
  <li>분전반/분전함 내부 부품 자동 인식</li>
  <li>이상 상태 자동 탐지</li>
  <li>부품별 명칭, 기능, 주의사항, 불량 가능성 안내</li>
  <li>문제 발생 시 조치 가이드 및 매뉴얼 제공</li>
  <li>점검 이미지 및 결과의 서버 자동 저장</li>
  <li>관리자용 점검 이력 조회 및 관리 기능 제공</li>
</ul>

<h2>🧩 대상 설비 및 구성 요소</h2>

<h3>1. 대상 설비</h3>

<ul>
  <li><b>배전반 (Switchboard)</b><br>
  비교적 큰 전력을 받아 여러 분전반으로 나누어 주는 대규모 설비</li>
  <li><b>분전반 / 분전함</b><br>
  배전반에서 받은 전력을 말단 회로 및 기기로 세분화하여 공급하는 설비</li>
</ul>

<h3>2. 주요 구성 요소</h3>

<ul>
  <li><b>차단기 (Circuit Breaker)</b>: 과부하·단락 등 이상 발생 시 전원을 차단하는 핵심 장치</li>
  <li><b>퓨즈 (Fuse)</b>: 과전류 발생 시 내부가 녹아 회로를 보호하는 부품</li>
  <li><b>배선 (Wiring)</b>: 전력을 전달하는 내부 전선 및 연결부</li>
</ul>

<h2>⚙️ 주요 기능</h2>

<h3>1. 사진 기반 설비 인식</h3>
<ul>
  <li>작업자가 분전반 또는 분전함 내부를 촬영</li>
  <li>QR 자동 인식 또는 수동 스캔으로 설비 식별</li>
  <li>설계도 및 기준 정보와 연동하여 분석 수행</li>
</ul>

<h3>2. 이상 탐지 기능</h3>
<ul>
  <li>차단기 ON/OFF 상태 감지</li>
  <li>단선 여부 감지</li>
  <li>부품 식별 및 부품별 매뉴얼 연결</li>
  <li>부식 감지</li>
  <li>먼지 누적 감지</li>
  <li>물방울 및 습기 흔적 감지</li>
  <li>번인 흔적 감지</li>
  <li>경고 스티커 / QR 라벨 훼손 또는 누락 감지</li>
  <li>빈 단자 존재 여부 확인</li>
</ul>

<h3>3. 결과 안내 기능</h3>
<ul>
  <li>정상 상태일 경우 점검 결과 요약 및 기본 매뉴얼 제공</li>
  <li>이상 상태일 경우 문제 부위, 원인 후보, 주의사항, 조치 매뉴얼 제공</li>
  <li>모든 결과를 이미지와 함께 서버에 자동 저장</li>
</ul>

<h2>🛠 시스템 구성</h2>

<h3>하드웨어</h3>
<ul>
  <li>Arduino (시각적 데모 효과용)</li>
  <li>Raspberry Pi (현장 추론 장치)</li>
  <li>카메라 (내부 부품 촬영 및 인식)</li>
  <li>LED 조명 (어두운 환경에서 가시성 확보)</li>
  <li>모바일-컴퓨터 연동 환경</li>
</ul>

<h3>소프트웨어</h3>
<ul>
  <li>YOLOv8 : 부품 및 이상 상태 인식 모델 학습</li>
  <li>Flask : 웹 서버 구축</li>
  <li>SQL : 점검 이력 및 진단 결과 저장</li>
  <li>Flutter / Dart : 모바일 및 웹 프론트엔드 구현</li>
  <li>Gemini API : 결과 설명, 매뉴얼, 요약 생성</li>
</ul>
