<h3>■ 팀 규칙</h3>
<br>1. 부득이하게 결석할 경우 사전에 팀원들에게 연락해야 하며 다른 날에 반드시 보강한다.
<br>2. 정기 모임은 매주 목요일 오전 10시 30분에 322호 리아 랩실에서 진행한다.
<br>3. 모든 팀원은 서로의 의견을 존중한다.
<br>4. 모든 의견 및 수정 사항은 발표 전날 또는 서류 제출 전날까지 통일하며, 전날 최종 점검 이후에는 새로운 의견 제시를 지양한다.
<br>5. 카톡 공지를 확인하면 반드시 확인 표 시 또는 답장을 남긴다. 질문이 있을 경우 반드시 답변해야 하며 무응답 시 전화로 확인할 수 있다.
<br>6. 추가 회의는 월요일 오후 2시-6시(상황에 따라 유동적으로 조정)에 진행한다.
<br>7. 팀의 최종 결정은 팀원 간 합의를 통해 결정한다.
<br>8. 서로 배려하고 협력하며 좋은 팀 분위기를 유지한다.
